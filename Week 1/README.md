⚰️ Proyecto Semana 1 – Ficha de Presentación de Dominio

Bootcamp JavaScript ES2023 – Semana 1

Este proyecto consiste en crear un script en JavaScript que se ejecute con Node.js y muestre en la consola una ficha informativa sobre una entidad de mi dominio asignado.

📋 Dominio del Proyecto

Dominio: Sistema de Gestión de Cementerios

Este dominio representa un sistema que permite registrar y administrar información relacionada con sepulturas, espacios disponibles y registros dentro de un cementerio.

La entidad principal utilizada en este proyecto es:

Entidad: Sepultura

🎯 Objetivo del Proyecto

El objetivo de este proyecto es practicar los conceptos básicos de programación en JavaScript utilizando:

console.log()

tipos de datos primitivos

comentarios en el código

operaciones aritméticas simples

El programa muestra una ficha informativa de una sepultura dentro del sistema de gestión del cementerio.

🧾 Información Mostrada por el Script

El script imprime en consola una ficha organizada que incluye:

Información general

Nombre de la sepultura

Código identificador

Categoría o tipo de sepultura

Estado actual

Detalles

Capacidad total de la sepultura

Cálculo de espacios disponibles

Estado de disponibilidad

Estado de mantenimiento

Resumen del dominio

Nombre del dominio

Tipo de entidad

Total de registros del sistema

JavaScript ES2023

Node.js

▶️ Cómo Ejecutar el Proyecto

Abrir la terminal.

Navegar hasta la carpeta del proyecto.

cd bootcamp/week-01/3-proyecto

Ejecutar el script con Node.js.

node starter/script.js

El programa mostrará en la consola la ficha de la sepultura registrada.

📊 Requisitos Cumplidos

Este proyecto cumple con los siguientes requisitos:

✔ Uso de console.log() para mostrar información
✔ Uso de strings para textos
✔ Uso de numbers para datos numéricos
✔ Uso de boolean para estados del sistema
✔ Uso de una operación aritmética para cálculo de espacios disponibles
✔ Comentarios en español explicando partes del código
✔ Salida organizada mediante separadores visuales

Entidad registrada en el sistema:

Nombre: Sepultura Familiar García

Código: CEM-245

Tipo: Sepultura familiar

Capacidad: 4 espacios

Espacios disponibles: 2

Este registro representa una sepultura dentro del sistema de gestión del cementerio.

Este proyecto permitió comprender los fundamentos básicos de JavaScript y cómo utilizar la consola para mostrar información estructurada.
