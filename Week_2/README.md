⚰️ Proyecto Semana 02 — Ficha de Datos del Dominio
📚 Bootcamp JavaScript ES2023

Este proyecto consiste en crear una ficha de datos en consola utilizando variables, tipos de datos y conversiones en JavaScript.

El programa se ejecuta con Node.js y muestra información organizada sobre una entidad perteneciente al dominio asignado.

🏛️ Dominio del Proyecto

Dominio: Sistema de Gestión de Cementerios

Este dominio representa un sistema encargado de administrar información sobre sepulturas, espacios disponibles, registros y estado de las áreas dentro de un cementerio.

La entidad utilizada para este proyecto es:

Entidad: Sepultura

🎯 Objetivo

Aplicar los conceptos aprendidos durante la Semana 02 del bootcamp, incluyendo:

Uso de variables con const

Uso de tipos de datos primitivos

Verificación de tipos con typeof

Conversión explícita de datos

Uso de nomenclatura correcta en variables

El script muestra una ficha informativa en la consola con datos relevantes del dominio.

📋 Datos Mostrados en la Ficha

El programa muestra la siguiente información:

Información principal

Nombre de la sepultura

Categoría o tipo

Capacidad de espacios

Estado de disponibilidad

Verificación de tipos

Se utilizan expresiones typeof para mostrar el tipo de datos de varias variables.

Conversión de datos

Se convierte un número a texto utilizando la función:

String()

Valor nulo

Se muestra un valor null que representa un dato que aún no ha sido asignado en el sistema.

▶️ Cómo Ejecutar el Proyecto

Abrir la terminal.

Navegar hasta la carpeta del proyecto.

cd bootcamp/week-02/3-proyecto

Ejecutar el script con Node.js.

node starter/script.js

El programa mostrará en la consola la ficha de datos correspondiente al dominio del sistema de gestión de cementerios.

📊 Requisitos Cumplidos

Este proyecto cumple con los requisitos establecidos:

✔ Uso de variables con const
✔ Uso de string, number, boolean y null
✔ Uso de typeof para verificar tipos de datos
✔ Conversión explícita de datos con String()
✔ Variables con nomenclatura camelCase
✔ Boolean con prefijo semántico (isItemAvailable)
✔ Script ejecutable sin errores con Node.js

📌 Conclusión

Este proyecto permitió reforzar los conceptos básicos de JavaScript relacionados con el uso de variables, tipos de datos y conversiones.

Además, se aplicó la adaptación del código a un dominio específico, en este caso un Sistema de Gestión de Cementerios, mostrando cómo organizar información de manera clara en la consola.
